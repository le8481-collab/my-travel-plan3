這個 App 使用 Firebase 雲端資料庫，讓你和朋友即時共用行程。
請按照以下步驟完成設定（只需一次）。

1
建立 Firebase 專案
前往 console.firebase.google.com，登入 Google 帳號後點「新增專案」，輸入專案名稱（例如 my-travel-app），其餘設定預設即可。
2
啟用 Realtime Database
在左側選單點「建構 → Realtime Database」→「建立資料庫」→ 選擇任一地區 → 安全性規則選「測試模式」（30天內免費開放讀寫）→ 啟用。
3
取得 Web 設定碼
回到專案總覽，點「</> 網頁」圖示新增 Web App，輸入暱稱後點「註冊應用程式」，然後複製 firebaseConfig 裡的各個值，填入下方欄位。
填入你的 Firebase 設定
